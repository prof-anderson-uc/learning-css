Cascading Style Sheets (CSS)

Style the Appearance of Elements and Control Page Layout

Cascading Style Sheets (CSS) is a coding language used for describing the visual presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS is designed to enable the separation of content (HTML) and presentation (CSS). Presentation includes page layout, colors, and fonts. This separation makes coding easier, provides more flexibility and control, enables multiple web pages to all have the same formatting by sharing a common .css file.

Separation of formatting and content also makes it easy to present the same page to appear differently for different devices, so your pages can look one way on a desktop but different on phones or tablets. This is called "responsive design" and is an increasingly important use of style sheets.

The "Cascading" part of CSS means that styles can fall (or cascade) from one style rule to another, enabling multiple rules to apply to a single element or even multiple style sheet files to be used on one HTML document. A child element will inherit the rules of its parent but you can overrule those styles by talking directly to the element itself.
