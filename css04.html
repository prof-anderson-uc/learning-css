Background Colors

CSS treats every HTML element as if it was inside a box, and the backgroud-color property gives that box an internal color. It will fill the entire inside of the box, including the padding, but stops at the border.

The default background color for elements is transparent but you can use CSS to give a color to anything you want. The two most common ways to add a background color is by using a color name or a hex value.

Background colors go great with block-level elements becaue they take up the entire width of their lines. They're also handy in container elements like section, div, header, nav, and footer.

In order to represent a color in CSS, you have to find a way to translate the analog concept of "color" into a digital form that a computer can use. This is typically done by breaking down the color into components, such as how much of each of a set of primary colors to mix together, or how bright to make the color. As such, there are several ways you can describe color in CSS.

How to Apply a Color

There are multiple ways of applyig a specific color in CSS. Two of the most popular methods are by using a color's pre-defined name or its hexidecimal value. The website htmlcolorcodes.com/color-names/ has a hany list of colors to choose from.

Color Names: There are 147 predefined color names that are recognized by browsers. Examples: Red, DarkGreen, Lavendar, LightSteelBlue

Hex Codes: These are 6-digit codes that represent the amount or red, green, and blue used to make a specific color, preceded by a # symbol. Examples: #E00122, #87CEEB, #CCCCCC
